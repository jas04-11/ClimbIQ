# 🐍 ClimbIQ: Snake & Ladder (AI Powered)

An interactive Snake & Ladder game built using Python Tkinter with a smart AI opponent powered by the Minimax algorithm.

---

## 🎯 Objectives

- Build an interactive GUI-based board game  
- Implement classic Snake & Ladder mechanics  
- Integrate AI using Minimax algorithm  
- Provide smooth animations and gameplay  
- Manage turns, states, and user interaction  
- Display real-time game updates and logs  

---

## 🎮 Features

- 🎲 Dice roll animation  
- 🤖 AI opponent (Minimax-based)  
- 🐍 Snakes & 🪜 Ladders logic  
- 📜 Move history log  
- 🌙 Dark-themed modern UI  
- ✨ Token animations & effects  

---

## 🧠 AI Implementation

The AI uses the **Minimax algorithm with Alpha-Beta pruning** to evaluate moves and select the best possible outcome, making the game more strategic and challenging.

---

## 🛠️ Tech Stack

- Python  
- Tkinter  
- Minimax Algorithm  

---

## 🎯 Gameplay Instructions

- Click **ROLL DICE** to play your turn  
- First player to reach **100 wins**  
- Avoid snakes and use ladders strategically  
- Compete against an intelligent AI
